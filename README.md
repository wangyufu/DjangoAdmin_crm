CRM
==
作者:王宇夫    

## 程序介绍:     
    使用python3+、django1.11，Windows或Linux环境使用；
	模仿django admin,几乎可以实现django admin所有功能；    
	实现了不同角色管理，不同的角色有不同的权限；
	细致的权限管理，可以将权限控制到是否允许一按键可以点击的级别
	

## 程序使用：     
	1、创建cmdb库，导入cmdb.sql      
	2、运行服务，游览器打开/web，主页    
	3、登录帐号 wangyufu@163.com/wangyufu1992
